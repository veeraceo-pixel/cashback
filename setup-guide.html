Step 1: Set Up Supabase Database

1

Create Supabase Account

  1. Go to supabase.com
  2. Click "Start your project"
  3. Sign up with GitHub or email
  4. Create a new organization (free tier is fine)
  5. Create a new project:
    • Name: dealcash-prod
    • Database Password: Save this securely!
    • Region: Choose closest to your users

Run Database Schema

  1. Wait for project to finish provisioning (2-3 minutes)
  2. Go to SQL Editor in the left sidebar
  3. Click "New query"
  4. Open your supabase-schema.sql file
  5. Copy all contents and paste into SQL Editor
  6. Click "Run" (bottom right)
  7. Verify: You should see "Success. No rows returned"

Add Sample Data

  1. Create another new query
  2. Open store-data-with-affiliate-links.sql
  3. Copy and paste contents
  4. Click "Run"
  5. Verify: Check Table Editor to see stores added

Get Your Credentials

  1. Go to Project Settings ‚Üí API
  2. Copy these values: 
    Project URL: https://xxxxx.supabase.co
Anon public key: eyJhbGc...
  3. Save these in a secure note

Step 4: Apply to Retailer Programs

4

Once approved by the networks, you need to apply to individual retailers:

In AWIN:

  1. Login to your AWIN account
  2. Go to "Advertisers" tab
  3. Search for stores (e.g., "Amazon", "ASOS")
  4. Click "Join Programme"
  5. Wait for approval (usually instant to 2 days)

Priority Retailers to Apply For:

  • Amazon UK (AWIN)
  • eBay UK (AWIN)
  • ASOS (AWIN)
  • Booking.com (AWIN)
  • Hotels.com (CJ)
  • Argos (AWIN)
  • Currys (AWIN)
  • John Lewis (AWIN)

Aim for 10-15 approved stores before launching.

Step 5: Get Your Affiliate Links

5

AWIN Links:

  1. Go to "Links & Tools" for each approved advertiser
  2. Copy your Publisher ID (e.g., 123456)
  3. Note the Advertiser ID (e.g., 2987 for ASOS)
  4. Build your tracking link: 
    https://www.awin1.com/cread.php?
  awinmid=ADVERTISER_ID
  &awinaffid=YOUR_PUBLISHER_ID
  &clickref={{CLICK_ID}}
  &p=DESTINATION_URL

Update Your Database:

UPDATE stores 
SET tracking_url = 'YOUR_ACTUAL_AFFILIATE_LINK'
WHERE slug = 'amazon-uk';
Important: Keep {{CLICK_ID}} in your links - this gets replaced dynamically!

Step 7: Create Admin Account

7
  1. Sign up on your website with your email
  2. Go to Supabase SQL Editor
  3. Run this query with YOUR email: 
    UPDATE auth.users 
SET raw_app_meta_data = raw_app_meta_data || '{"role": "admin"}'::jsonb
WHERE email = 'youremail@example.com';
  4. Log out and log back in
  5. Access admin panel at /admin.html

Step 8: Testing

8

Test Checklist:

  • Sign up works and sends confirmation email
  • Login works
  • Stores display on homepage
  • Clicking "Shop Now" tracks click in database
  • User is redirected to retailer
  • Admin can view clicks in admin panel
  • Make a small test purchase (if possible)
  • Verify transaction appears in database

How to Test Without Spending Money:

  1. Add a test transaction manually in database: 
    INSERT INTO transactions (
    user_id, store_id, click_id,
    order_amount, cashback_amount, status
) VALUES (
    'user-uuid', 'store-uuid', 'test-click-123',
    100.00, 5.00, 'confirmed'
);
  2. Check if it appears in user dashboard
  3. Test withdrawal request

Step 9: Set Up Webhooks (Advanced)

9

Webhooks automatically update transactions when retailers confirm purchases.

  3. Configure in affiliate networks:
    • AWIN: Settings ‚Üí Postback URL
    • CJ: Account ‚Üí Tracking Integration
    • ShareASale: Tools ‚Üí Postback

‚ùì Troubleshooting

Tracking Not Working?

  • Check browser console for JavaScript errors
  • Verify Supabase credentials are correct
  • Ensure click_id is being generated
  • Test affiliate links manually

Users Can't Sign Up?

  • Check Supabase Auth settings
  • Verify email confirmation is enabled
  • Check spam folder for confirmation emails

Cashback Not Showing?

  • Check transaction status in database
  • Verify triggers are enabled
  • Manually update status to test
